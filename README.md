# digit-recognition-cnn1
CNN-based handwritten digit recognition using the MNIST dataset with a Streamlit web application built using TensorFlow and Keras. 🚀
# Handwritten Digit Recognition

This project uses a CNN trained on the MNIST dataset.

## Technologies

- Python
- TensorFlow
- Streamlit
- NumPy
- Pillow

## Run

pip install -r requirements.txt

streamlit run app.py
